# CrabThread

CrabThread is a fast, multi-threaded Rust web server designed to handle concurrent requests efficiently.

## Usage

```bash
# Run the server
cargo run --release
````

By default, the server listens on `127.0.0.1:7878`. You can change the port in the code if needed.

## Installation

```bash
git clone https://github.com/yourusername/crabthread.git
cd crabthread
cargo build --release
```
